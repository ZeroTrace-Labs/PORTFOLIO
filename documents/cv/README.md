# CV Folder

This folder stores your professional CV for download.

## Instructions

1. **Upload your CV**: Place your CV PDF file in this folder
2. **File naming**: Name the file `Morgan-Mutugi-CV.pdf` (matches the download button reference)
3. **Download functionality**: When users click the "Download Professional CV" button, they will download this PDF file

## Expected File

- **Filename**: `Morgan-Mutugi-CV.pdf`
- **Format**: PDF
- **Location**: `/documents/cv/Morgan-Mutugi-CV.pdf`

Once you upload the CV PDF file to this folder, the download button will work automatically.
